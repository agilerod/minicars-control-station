# MiniCars Control Station Desktop (Vite + React + TS)

## Instalación

```bash
cd desktop
npm install
```

## Modo desarrollo

```bash
npm run dev
```

> Asegúrate de que el backend FastAPI esté corriendo. Por defecto, la UI se conecta a `http://127.0.0.1:8000`, pero puedes configurarlo mediante la variable de entorno `VITE_MINICARS_BACKEND_URL` en `desktop/.env` (ver `desktop/.env.example`).
